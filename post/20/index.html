阿里云轻应用服务器安装宝塔面板

Posted on 2019.09.03，3 min read

今天通过阿里云开发者社区的开发者认证测试，就能获得一张优惠券阿里云开发者认证活动地址。我也去撸了一张200元的优惠券，买了一个阿里云香港的轻应用服务器，想着在上面安装一下宝塔面板，下面记录一下小白安装宝塔的过程。

购买轻应用服务器

我购买的是最低的配置，CPU1核，内存1G，25G SSD。搜索了一下，网上说阿里云提供的LNMP有一些小问题，所以我打算安装CentOS，然后再通过宝塔面板安装LNMP。

设置服务器密码

点击重置密码设置服务器的密码，后面安装的时候要用。

开放服务器端口

点击左侧安全>防火墙，点击“添加规则”开放以下端口

HTTP TCP 80
HTTPS TCP 443
SSH TCP 22
自定义 TCP 8888
自定义 TCP 20
自定义 TCP 21
自定义 TCP 3306
自定义 TCP 888
自定义 TCP 39000/40000

安装宝塔面板

点击右上角的远程连接，开始安装宝塔面板（宝塔面板官方安装教程）。

以管理员身份运行以下命令,若没有以管理员身份运行下面命令，会出现you need to be root to perform this command提示，这个时候输入su 回车输入密码，即可解决。

Centos安装命令：

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

等待安装完成，注意在安装过程中需要输入y来确认。

等到出现如下的界面，则说明已经安装成功，这个时候，我们把图中的宝塔地址和用户名以及密码保存下来，

登录到宝塔面板后台，安装web环境

选择推荐的就好，点击一键安装，然后静待安装完成即可。

连接FTP遇到的问题

1.FTP连接报错 421 Unable to read the indexed puredb file (or old format detected) - Try pure-pw mkdb

解决：在宝塔后台FTP管理中添加一个用户，同时修改ftp配置文件 将ForcePassiveIP前面的#去掉 将192.168.0.1替换成服务器的外网IP。更多见 FTP连接不上的解决方法
